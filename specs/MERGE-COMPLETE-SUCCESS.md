### **Backend:**
```bash
cd chaingive-backend

# Install dependencies
npm install

# Run migrations
npx prisma generate
npx prisma migrate deploy
```

```bash

# Install dependencies

# iOS setup

# Edit API_BASE_URL

# Run
npm run ios
# or
npm run android
```
